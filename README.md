# 👀 DevEyes

> Code Smarter. Blink Better. Stay Healthy.

DevEyes is a Visual Studio Code extension that helps developers reduce digital eye strain and build healthier coding habits. It provides smart eye-care reminders, guided eye exercises, progress tracking, and an interactive wellness dashboard—all without leaving VS Code.

---

## ✨ Features

- 👀 Smart eye-care reminders
- ⏰ 20-20-20 rule notifications
- 🎯 Random eye exercises
- 📊 Daily wellness statistics
- 📈 Progress dashboard 
- 🔥 Daily streak tracking 
- 🏆 Achievements 
- 💧 Water reminders 
- 🧘 Stretch reminders 
- ⚙️ Custom reminder intervals 



# 🚀 Installation
